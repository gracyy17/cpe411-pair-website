Frequently Asked Questions

Quick answers about this CPE 411 pair website: what it contains, how to navigate it, and how to contact the team.

Questions

Click a question to expand the answer.

What is this website for?

This is a CPE 411 pair website that presents our work and introduces the team. It includes a homepage, a Projects page, and an About page.

Who made this website?

The site was created by Kenneth Sarmiento, John Joseph Manalang, and Hellene Grace Septimo. See the About page for team details.

Where can I find your projects and deliverables?

Go to the Projects page. It summarizes what we built and what each part contains.

How do I navigate the site?

Use the top navigation bar to switch between pages (Home, Projects, About, FAQ). Each page keeps the same header and footer for consistency.

Is the website mobile-friendly?

Yes. The layout uses responsive CSS so sections and cards stack nicely on smaller screens.

What technologies did you use?

The site is built with HTML and CSS. We used semantic structure (headings, sections, navigation) and a shared theme using CSS variables.

How can I contact the team?

Visit the Contacts section on the About page to email us.

I found a typo or layout issue—what should I do?

Please message any team member through the emails listed on the About page and include: the page name, what you expected, and (if possible) a screenshot.
