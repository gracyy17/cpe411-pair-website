Website Build (Pair Website)

A small multi-page website that presents the team and provides a clean, consistent navigation experience.

Overview

The goal was to create a readable and modern-looking site with a shared theme, while keeping the HTML semantic and the layout mobile-friendly. We focused on consistency (same header/footer), clarity (short sections), and accessibility (proper headings and navigation labels).

Key features

  • Sticky navigation header with clear page links.
  • Consistent theme using CSS variables.
  • Responsive layouts (works on phones and desktops).
